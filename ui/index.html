SealSounds

A cleaner way to listen to albums. No ads, no comments, no commercial breaks — just music, art, and context.

About

SealSounds plays full albums from YouTube in a distraction-free environment, enriched with artist photos and facts from open sources. All visual content is sourced from public domain and Creative Commons licensed material via Wikimedia Commons and Wikipedia.

Version History

v1.1.14
Version history overlay (H key). Volumetric clouds (key 8) and Highland landscape (key 9) visualizations with adaptive resolution targeting 20 FPS. Search enter fix when returning from album. Removed search input clear button.
v1.1.13
Security hardening: XSS prevention (innerHTML replaced with textContent), Content Security Policy headers for web and Tauri, 10-second fetch timeouts on all API calls, localStorage cache pruning. Wikimedia image filtering to exclude medical/scientific content. Search input clear button removed.
v1.1.12
Mobile "Did you know" facts expanded to 3 lines. Bottom content area enlarged to 155px.
v1.1.11
Background images now visible on mobile with slideshow mode. Mandelbrot auto-activates when no images found, cycles every 3 minutes between images and Mandelbrot when images available.
v1.1.10
"Back to album" button appears on search screen when music is playing, allowing return without re-searching.
v1.1.9
Mobile nav redesigned: back arrow on left, info (i) button on right, flanking centered YouTube player. Buttons reduced to 44px for better fit.
v1.1.8
Mandelbrot zoomer cycles through 6 interesting locations (Seahorse Valley, Spiral Arm, Elephant Valley, Double Spiral, Mini Mandelbrot, Julia Cusp) every 90 seconds with crossfade transitions.
v1.1.7
Mobile nav buttons visible whenever in listening mode (not waiting for autoplay). Smoke effect more visible on mobile search screen. Info shortcuts reordered.
v1.1.6
Visualization manager with 7 effects selectable via keys 1-7, S to cycle, 0 for slideshow. Added VU Meters, Spectrum Analyzer, Starfield, Butterflies (3D raymarched), Aurora City, and Rainforest. Album select button (Q / Arrow Up). VU meters and spectrum freeze when paused. Console API logging.
v1.1.0
Album loops when reaching end. Search changed to Enter/submit only (saves ~80% API quota). localStorage caching with 24-hour TTL. Hidden test mode ("testalbum"). Swipe left/right for track navigation. Mobile tracklist vertically centered and semi-transparent. Mandelbrot zoom speed increased.
v1.0.0
First release. Modular ES6 architecture (player, search, controls, tracklist, images, facts, utils). Plugin system with capability detection. Ambient smoke effect. Mandelbrot zoom. Mobile responsive layout. Test suite (44 tests). Config separated from code.
v0.10.2
Initial commit. YouTube album search and playback, Wikipedia artist facts, Wikimedia Commons images with transitions, tracklist parsing from descriptions and comments, keyboard shortcuts, desktop Tauri wrapper.
